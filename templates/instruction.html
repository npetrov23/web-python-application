Для того, чтобы оценка формировалась автоматически необходимо настроить форматирование оценок.
1. Выберите нужный статистический показатель из выпадающего списка



2. Выберите категорию из выпадающего списка



3. Настройте верхнюю и нижнюю границы для пяти видов оценок
Верхняя и нижняя граница образуют диапазон чисел от X до Y, где Х - верхняя граница, Y - нижняя граница (включительно)



Обратите внимание:
1. Форматирование для оценки не определено означает, что форматирование не было настроено. Для этого заполните поля ниже
2. Выход за указанный диапазон означает, что значение статистического показателя вышло за пределы указанных границ. Для изменения границ обратитесь к администратору

Для того чтобы занести в систему показатели спортсмена необходимо нажать на кнопку "Добавить показатели"



В открывшемся окне выберите нужного Вам спортсмена



Затем выберите дату норматива (1, 2) и заполните поля показателями спортсмена (3). Нажмите на кнопку "Сохранить изменения" (4)



Чтобы просмотреть введеные данные выберите из выпадающего меню нужного Вам спортсмена (1), дату норматива (2) и категорию (3)
Нажмите на кнопку "Вывести", после чего в колонке "Результат" (4) отобразятся заполненные Вами данные



Чтобы посмотреть динамику результатов спортсмена Вы можете построить графики. Для этого нажмите на кнопку "Построить графики"



На открывшейся странице выберите нужный тип подготовленности (1), укажите спортсмена (2) и диапазон времени, за который Вы хотите
посмотреть результаты (3). Чтобы вывести графики нажмите на кнопку "Визуализировать" (4).



Вы можете распечатать отдельный график или сохранить его в виде изображения. Для этого нажмите на кнопку гамбургера и во всплывающем меню выберите
подходящий для Вас вариант



Спортсмены не могут регистрироваться в системе самостоятельно. Зарегистрировать их должен тренер. Для этого нажмите на кнопку "Добавить спортсмена"



В открывшемся окне введите английскими буквами логин спортсмена (1). Он будет использоваться для входа в систему. Укажите имя и фаилию спортсмена (2)
Введите адрес электронной почты (3). Он необходим для восстановления пароля. Заполните поля с паролем (4), введите капчу (5) и нажмите на кнопку
"Добавить спортсмена" (6). После этого в системе будет создана учетная запись спортсмена. Сообщите ему данные для входа



Данная страница содержит инструкцию по основным моментам работы с системой.
